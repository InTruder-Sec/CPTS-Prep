# Nmap

Nmap Scripts:  `/usr/share/nmap/scripts/` ,  `nmap --script <script name> -p<port> <host>`

Banner Grabbing: `nmap -sV --script=banner <target>`

1. Most basic scan

```bash
nmap 10.129.42.253
```

1.  `-sC` parameter to specify that `Nmap` scripts should be used to try and obtain more detailed information. The `-sV` parameter instructs `Nmap` to perform a version scan

